## Description

This is [@ts-stack/mod](https://github.com/ts-stack/mod) seed project.

## Usage

From first terminal:

```bash
git clone git@github.com:ts-stack/mod-seed.git
cd mod-seed
npm i
npm start
```

From second terminal, check work:

```bash
curl -isS localhost:8080
curl -isS localhost:8080/throw-error
```
